Playfair

The Playfair Cipher encrypts pairs of letters using a 5x5 matrix built from a keyword. It replaces each pair according to rules involving their positions in the grid. If a pair shares a row, column, or forms a rectangle, specific swapping rules are applied.

Vigenere

The Vigenère Cipher uses a repeating keyword to determine the shift for each letter. Each letter in the plaintext is shifted according to the corresponding letter of the keyword, creating a polyalphabetic substitution.
